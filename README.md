# Glass Fracture AI Solver 🔬

An AI-powered physics engine that uses **Inverse Neural Networks** to predict glass fracture patterns and calculate optimal drop conditions.

## 🚀 Live Demo
[Check out the interactive tool on Hugging Face](আপনার_হাগিং_ফেস_লিঙ্ক_এখানে_দিন)

## 🛠️ Features
- **Inverse Solving:** Calculates Height and Angle based on Target Distance.
- **AI Model:** Built using PyTorch and Scikit-learn.
- **Interactive UI:** Simple web interface using Gradio.
